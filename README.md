# Trader Performance vs Market Sentiment Analysis

## Methodology

- Loaded the Bitcoin Fear & Greed Index and Hyperliquid trader datasets.
- Cleaned the data by checking for missing values and duplicates.
- Converted timestamps to a daily date format and merged the datasets using the Date column.
- Engineered features including daily PnL, win rate, average trade size, leverage, trade frequency, and long/short ratio.
- Compared trader performance and behavior across Fear and Greed market conditions.
- Segmented traders based on leverage, trading frequency, and consistency.
